# Next-Bet-AI (v2.2-Beta) 🚀

Plateforme de prédiction de football basée sur le Deep Learning et l'architecture Neural Network.
Analyses en temps réel pour la Ligue 1 et la Premier League.

## 🔐 Système de Bêta Privée
Cette application est actuellement en **Bêta Privée**. 
1. Inscrivez-vous sur l'interface.
2. Votre accès sera validé manuellement par l'administrateur.

## 🛠️ Stack Technique
- **Backend** : FastAPI (Python)
- **Modèle IA** : PyTorch (MatchPredictor v2)
- **Frontend** : Next.js 14
- **Base de données** : Supabase Cloud
- **Paiements** : Stripe Integration

## 📦 Déploiement
Ce dépôt contient le Dockerfile optimisé pour **Hugging Face Spaces**.
